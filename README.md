minesweeper-gtk
================

## about

Classic minesweeper game written on C++.
Uses gtkmm 2.x for UI.
Original forked from mlfong/minesweeper-gtk

## build instructions

    $ git clone git@github.com:nitrocaster/minesweeper-gtk.git
    $ make

## run instructions
    
    $ ./Minesweeper

or double-click the generated executable.
